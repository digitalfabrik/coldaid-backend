generate_route()¶

generates a url that is conform to google maps api. When the url is clicked, the google maps app will open (resp. a browser tab on non-mobile devices) and a route based on all requests is shown that: 1. are assigned to this vehicle object and 2. have the activeRoute field set to true (TODO pre-specify a sequence that is used to generate the route here) The url is generated from the given address, postcode and city fields of each request :return: url containing als selected requests
