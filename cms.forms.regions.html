cms.forms.regions.region_form.duplicate_language_tree(source_region, target_region, source_parent_id=None, target_parent=None)¶

Function to duplicate the language tree of one region to another.

Usage: duplicate_language_tree(source_region, target_region)

This is a recursive function to walk the whole language tree. It starts at root level with the default parent None. The recursion is necessary because the new nodes need their correct (also duplicated) parent node.

Args:
param source_region

The region from which the language tree should be duplicated

param target_region

The region to which the language tree should be added

param source_parent_id

The current parent node id of the recursion

param target_parent

The node of the target region which is the duplicate of the source parent node
