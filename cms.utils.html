Utils¶

This package contains utilities which faciliate specific tasks and can be re-used and imported from different locations.

File Utils¶

This module contains helpers for file handling.

cms.utils.file_utils.save_file(request)[source]¶

This function accepts uploaded files, checks if they are valid in respect to the DocumentForm and stores them to disk if so.

Example usage: cms.views.media.media_edit_view.MediaEditView

Parameters:request (django.http.HttpRequest) – The current request submitting the file(s)
Returns:A dictionary containing the DocumentForm object and the boolean return status
Return type:dict

MFA Utils¶

This module contains helpers for multi-factor-authentication tasks.

cms.utils.mfa_utils.generate_challenge(challenge_len)[source]¶

This function generates a random challenge of the given length. It consists of ascii letters and digits.

Example usage: cms.views.settings.mfa.mfa.GetChallengeView

Parameters:challenge_len (int) – The desired length of the challenge
Returns:A random string of the requested length
Return type:str

Slug Utils¶

This module contains helpers regarding unique string identifiers without special characters (“slugs”).

cms.utils.slug_utils.generate_unique_slug(form_object, foreign_model)[source]¶

This function can be used in forms to clean slug fields. It will make sure the slug field contains a unique identifier per region and language. It can also be used for region slugs (foreign_model is None in this case). If the slug field is empty, it creates a fallback value from either the title or the name field. In case the slug exists already, it appends a counter which is increased until the slug is unique.

Example usages:

  • cms.forms.regions.region_form.RegionForm.clean_slug()
  • cms.forms.pages.page_translation_form.PageTranslationForm.clean_slug()
  • cms.forms.events.event_translation_form.EventTranslationForm.clean_slug()
  • cms.forms.pois.poi_translation_form.POITranslationForm.clean_slug()
Parameters:
  • form_object (django.forms.Form) – The form which contains the slug field
  • foreign_model (django.db.models.Model) – If the form instance has a foreign key to another model (e.g. because it is a translation of a content-object), this paramaeter contains the model of the foreign related object.
Returns:

An unique slug identifier
Return type:

str
